============================================================
           Easy Tooltip v1.0 by Ahmed Benlakhdhar
============================================================

A simple, zero-setup tooltip system for Unity UI.


---------------------------------
:: Quick Start ::
---------------------------------

1.  Add the "TooltipTrigger" component to any UI element (Button, Image, etc.).
2.  Fill in the Title and Content fields in the Inspector.

Done. The manager is created automatically.
(See the Demo Scene for more examples).


---------------------------------
:: Features ::
---------------------------------

• Zero Setup Required
• Rich Content (Title, Content, Icon)
• Custom Styles (Title & Icon Colors)
• Smart Text Wrapping with Max Width
• Smooth Fade Animations
• Stays On-Screen Automatically


---------------------------------
:: Configuration ::
---------------------------------

You can configure global settings (Max Width, Fade Speed, etc.) in two ways:

1.  **Global Settings (Recommended):**
    - Edit the "TooltipManager" prefab directly. You can find it in your project's Resources folder at: `Assets/Easy Tooltip/Resources/`.

2.  **Per-Scene Overrides (Optional):**
    - If you need different settings for a specific scene, you can drag the "TooltipManager" prefab into that scene's hierarchy. The system will automatically use the settings from that instance for that scene only.
